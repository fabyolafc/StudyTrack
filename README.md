- 🌐 **Frontend:** https://study-trackk.vercel.app

---

## 🧠 Sobre o Projeto

O **StudyTrack** é uma aplicação desenvolvida para ajudar estudantes a organizarem sua rotina de estudos, criando metas e registrando o progresso ao longo do tempo.

Com uma interface moderna e intuitiva, o sistema permite visualizar o desempenho através de gráficos 📊.

---

## ✨ Funcionalidades

### 🔐 Autenticação
- Cadastro de usuário
- Login com JWT
- Persistência de sessão

### 🎯 Metas
- Criar metas de estudo
- Editar metas
- Excluir metas
- Listar metas

### 📚 Estudos
- Registrar horas estudadas
- Associar estudo a uma meta
- Editar estudos
- Excluir estudos

### 📈 Dashboard
- Visualização de progresso
- Gráficos dinâmicos
- Interface moderna
